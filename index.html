My Project

Firebase Chat App

The main objective of this paper is to present a software application for the launching fo a real time communication between operators/users. The system requires both the device to be connected via internet. This application is based on Android with the backend provided by google Firebase.

  • Introduction - Chat Application
    Communication

    Firebase provides a various kind of services includes Chat application many-to-many type of communication system where the users will able to exchange the messages among themselves. User can create the chatroom according to the requirement or can also join to the existing chatroom. The basic goal is to create a system which can create chatroom according to the users and store the related data at a single place.

  • Proposed System -Advantages
    PDF Creation

    By using this system, the user need not have any other pdf reader-writer application on the device. In this application we are using Google firebase as the backend to store the data of the application such as messages, pictures, files and more. There are many other chat applications like WeChat, WhatsApp, Telegram, Facebook messenger, Snap Chat, Line etc but in this system the pdf creation and pdf reading feature will be included.

  • Front-End -Software Requirements
    Mobile And Android

    • IDE : Android Studio
    • Languages : Java & XML
    • SDK : Firebase

  • Back-End - Software Requirements
    Firebase - Realtime Database

    The Firebase Realtime Database is a cloud-hosted database. Data is stored as JSON and synchronized in realtime to every connected client. When you build cross-platform apps with our Apple platforms, Android, and JavaScript SDKs, all of your clients share one Realtime Database instance and automatically receive updates with the newest data.

  • Modules -Application
    Mobile Friendly

    • User Login
    • User Registration
    • profile
    • Group Chat
    • One to One Chat
    • PDF Creation
    • Send Files

  • Conclusion -Project

    The final system will result as a real time communication application which provides the users to communicate to each other with an ease. The application will have a login page through which the user can register and login themselves. Home page of the application contains the previous messages if any.
